Lista com comandos básicos do git

  • git init - Cria um repoitório em uma pasta vazia.
  • git add - Adiciona modificações do wk no stage
  • git status - Mostra o estado atual do repoitório, modificações arquivos versionados
  • git commit - Cria um commit no repoitório local
  • git push - Envia as modificações do repoitório local para o remoto
  • git log - Exibe no terminal os commitsdo repoitório na branch atual
  • git log --oneline --all - Exibe todos os commits de forma resumida
  • git brach - Lista as branches do repoitório
  • checkout - Muda de branch
  • checkout -b - Cria e muda para uma nova brach
